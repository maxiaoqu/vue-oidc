## 使用技术

- vue
- vue-router
- oidc-client
- vue-cli3
- TypeScript
- Less\css

## 安装

```
npm install
```

## 运行

```
npm run serve
```

## 打包

```
npm run build
```

## 打包分析

```
npm run build --report
```

## 检查

```
npm run lint
```

## 文件结构

```shell
├── public...........................打包所需静态资源
└── src..............................存放文件的相关目录
    ├── method.......................相关的方法
    ├── router.......................路由文件
    ├── store........................vuex状态管理
    ├── views........................vue页面文件
    ├── APP.vue......................app.vue文件
    └── main.js......................vue相关全局配置
```
